Zilliqa Staking Caculator

How to embed

Clone the project:

git clone https://github.com/Zilliqa/staking-calculator-plugin.git

Copy the .css and .js files and paste the following code blocks to the head and body tags:

            <head>
              ...
              <link rel="stylesheet" type="text/css" href="./zilliqa-staking-calculator.css"/>
            </head>
        
            <body>
              ...
              <div id="staking-calculator"></div>
              <script src="./zilliqa-staking-calculator.js"></script>
            </body>
